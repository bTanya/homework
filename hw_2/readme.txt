Реализация треугольника Серпинского
Начинаем с большого треугольника, который делим на четыре маленьких, связанных с серединами сторон первоначального.
Игнорируя вновь созданный внутренний треугольник, делаем всё то же самое для каждого из трёх угловых.
Каждый раз при создании нового набора треугольников,  рекурсивно применятся эту процедуру к трём меньшим угловым фигурам.
